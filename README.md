# 3X-UI Glass

A customized 3X-UI panel with Glassmorphism UI theme.

## Features
- 🎨 Glassmorphism UI (dark/light/ultra-dark mode)
- 📊 Expandable clients under each inbound
- 📱 QR Code with copy support
- 🔒 IP Limit per client
- 🛡️ SSL Certificate (Let's Encrypt)
- 📦 Offline installer

## Installation

Download the latest release and transfer to your server:

```bash
wget https://github.com/alir3zai/3x-ui-glass/releases/download/v3.4/3x-ui-glass.tar.gz
tar xzf 3x-ui-glass.tar.gz
cd 3x-ui-glass
bash install.sh
```
